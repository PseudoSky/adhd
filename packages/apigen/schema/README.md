# @adhd/apigen-schema

JSON Schema support for the apigen toolchain. Pure TypeScript (**platform: shared**).

Part of [apigen](../README.md). For end-to-end usage see [`../cli`](../cli).

> Minimal/reserved package. The canonical type IR (JSON Schema 2020-12), extraction, and the
> validation Layer live in [`@adhd/apigen-core`](../core) and [`@adhd/apigen-runtime`](../runtime).
